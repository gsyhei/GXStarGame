# GXStarGame - 消消乐游戏

一个精美的iOS消消乐游戏，具有3D视觉效果和爱心图案。

## 游戏特色

### 🎨 视觉效果
- **圆角设计**: 所有方块都采用圆角设计，看起来更加现代
- **3D效果**: 每个方块都有阴影和渐变效果，营造立体感
- **爱心图案**: 每个方块中心都有一个精美的白色爱心图标
- **动画效果**: 包含选中、匹配、消除、下落等多种流畅动画

### 🎮 游戏玩法
- **经典消消乐**: 通过交换相邻方块来形成3个或更多相同颜色的连线
- **智能匹配**: 自动检测水平和垂直方向的匹配
- **连锁反应**: 消除后自动下落新方块，可能产生新的匹配
- **分数系统**: 每消除一个方块获得10分

### 🎯 游戏规则
1. 点击选择一个方块
2. 点击相邻的方块进行交换
3. 只有能形成匹配的交换才会被允许
4. 形成3个或更多相同颜色的连线即可消除
5. 消除后上方方块会下落填补空隙
6. 新方块会从顶部生成

## 技术实现

### 架构设计
- **MVC模式**: 清晰的模型-视图-控制器分离
- **委托模式**: 游戏引擎与视图控制器通过委托进行通信
- **组件化**: 每个功能模块独立实现

### 核心组件
- `GameModels.swift`: 游戏数据模型和配置
- `GemView.swift`: 带3D效果的方块视图
- `GameEngine.swift`: 游戏逻辑引擎
- `ViewController.swift`: 主游戏界面控制器

### 视觉效果实现
- **渐变背景**: 使用CAGradientLayer创建渐变效果
- **阴影效果**: 通过layer.shadow属性实现3D阴影
- **爱心绘制**: 使用Core Graphics绘制数学公式爱心
- **动画系统**: 基于UIView动画的流畅过渡效果

## 运行要求
- iOS 18.5+
- Xcode 16.0+
- Swift 5.0+

## 安装运行
1. 使用Xcode打开 `GXStarGame.xcodeproj`
2. 选择目标设备或模拟器
3. 点击运行按钮或按 `Cmd+R`
