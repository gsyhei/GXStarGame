# 隐私清单说明 (Privacy Manifest)

## 文件位置
`GXStarGame/GXStarGame/PrivacyInfo.xcprivacy`

## 清单内容

### 1. 追踪说明
- **NSPrivacyTracking**: `false`
  - 应用不进行用户追踪
  - 不使用IDFA或其他追踪标识符
  - 不与第三方共享用户数据用于广告目的

### 2. 数据收集
- **NSPrivacyCollectedDataTypes**: 空数组
  - 应用不收集任何用户数据
  - 不上传用户信息到服务器
  - 所有数据仅存储在本地设备

### 3. API使用声明

#### UserDefaults (CA92.1)
- **用途**: 存储游戏设置（音乐/音效开关）
- **理由代码**: CA92.1 - 访问应用偏好设置
- **具体用途**:
  - 保存音乐开关状态 (isMusicEnabled)
  - 保存音效开关状态 (isSoundEffectsEnabled)
  - 首次启动标记 (hasLaunchedBefore)
  - 数据不会离开用户设备

#### File Timestamp (C617.1)
- **用途**: 系统文件访问
- **理由代码**: C617.1 - 由系统或应用自动管理的文件访问
- **具体用途**:
  - 访问应用包内的资源文件（音频、图像等）
  - 正常应用功能所需

### 常见问题

**Q: 为什么需要PrivacyInfo.xcprivacy？**
A: 从iOS 17开始，Apple要求所有使用特定API的应用都必须包含隐私清单，说明API的使用目的。

**Q: CA92.1是什么意思？**
A: 这是Apple定义的API使用理由代码，CA92.1表示"访问用户偏好设置"，用于应用功能所需的配置存储。

**Q: 如果审核被拒绝怎么办？**
A: 
1. 查看拒绝原因
2. 如需添加其他隐私声明，可以更新PrivacyInfo.xcprivacy
3. 确保所有声明与实际代码一致
