## 🎹 使用 GarageBand 快速制作（Mac）

### 制作背景音乐

1. **打开 GarageBand**
   - Mac自带应用
   - 选择"新建项目" → "空项目"

2. **选择乐器**
   - 软件乐器 → 键盘
   - 选择"Dance"或"Electronic"分类
   - 选择轻快的音色

3. **创作循环**
   - 长度：30-60秒
   - 选择合适的速度（BPM: 120-140）
   - 使用循环片段快速创作

4. **导出**
   - 文件 → 导出歌曲到磁盘
   - 格式：MP3
   - 音质：128 kbps
   - 重命名为：`background_music.mp3`

### 制作音效

1. **使用GarageBand的音效库**
   - 循环浏览器 → 音效
   - 搜索："Click", "Pop", "Bell" 等

2. **导出单个音效**
   - 创建短项目（1-2秒）
   - 添加音效
   - 导出为MP3

## 📦 打包下载建议

### 创建音频文件夹

在桌面创建文件夹：`GXStarGame-Audio`

```bash
mkdir ~/Desktop/GXStarGame-Audio
```

### 下载并重命名

下载音频后，重命名为项目需要的文件名：

```
GXStarGame-Audio/
├── background_music.mp3       ← 背景音乐
├── select.mp3                 ← 选择音效
├── swap.mp3                   ← 交换音效（可选）
├── match.mp3                  ← 消除音效
├── drop.mp3                   ← 下落音效（可选）
├── game_over.mp3              ← 游戏结束
└── button_tap.mp3             ← 按钮点击
```

### 批量添加到Xcode

1. 选中所有音频文件
2. 拖入Xcode项目
3. 确认勾选"Copy items if needed"
4. 完成！

## ⚖️ 版权安全建议

### 最安全的选择
1. **CC0授权** - 公共域，完全免费使用
2. **付费购买** - 获得商业使用授权
3. **自己制作** - 完全拥有版权

### 需要注意
1. **CC BY授权** - 需要署名，但可商用
2. **免费但禁止商业使用** - 不适合App Store应用
3. **YouTube Audio Library** - 注意查看具体授权

### 在游戏中署名（如果需要）

在设置界面的"关于"部分添加：

```swift
// 在 SettingsViewController 中添加
let creditsLabel = UILabel()

Music: "Song Name" by Artist
License: CC BY 3.0
"""

creditsLabel.textColor = .secondaryLabel
```
