## 🎮 完整游戏流程

### 1. 启动应用
- 显示启动页（紫色渐变 + logo）
- 加载游戏资源
- 初始化音频系统

### 2. 主菜单
- 进入主菜单界面
- 自动播放背景音乐
- 图标跳动动画
- 可以选择：
  - 🎮 开始游戏
  - ⚙️ 设置

### 3. 开始游戏
- 点击"开始游戏"（音效）
- 进入游戏界面
- 背景音乐继续播放
- 开始游玩

### 4. 游戏进行中
- 选择宝石 → 音效
- 消除匹配 → 音效
- 分数更新 → 动画
- 可以：
  - 🔄 重新开始
  - ⬅️ 返回菜单

### 5. 游戏结束
- 触发结束条件
- 播放游戏结束音效
- 显示最终分数
- 可以重新开始或返回菜单

### 6. 设置
- 从菜单进入设置
- 调整音乐/音效开关
- 查看关于信息
- 联系技术支持
- 设置自动保存
