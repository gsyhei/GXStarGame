## 🎓 使用说明

### 快速开始
1. 用Xcode打开项目
2. 选择模拟器或真机
3. 点击运行 ▶️

### 生成App Icon
在 `AppDelegate.swift` 中：
```swift
#if DEBUG
AppIconGenerator.generateIcons()
#endif
```

### 测试不同语言
在Xcode中：
1. Product -> Scheme -> Edit Scheme...
2. Run -> Options -> App Language
3. 选择 Chinese, Simplified 或 English
4. 运行测试

### 配置Bundle ID
1. 选择项目 -> Target -> General
2. Identity -> Bundle Identifier
3. 输入：com.你的公司名.StarMatch
