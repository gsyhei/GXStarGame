## 🚀 快速开始（10分钟完成）

### 步骤1：启动模拟器并运行游戏

```bash
# 1. 进入项目目录
cd /Users/gin/CursorWorkspace/GXStarGame

# 2. 打开Xcode项目
open GXStarGame/GXStarGame.xcodeproj
```

在Xcode中：
1. 选择模拟器：**iPhone 15 Pro Max**（6.7英寸）
2. 点击运行 ▶️
3. 等待应用启动

## 📤 上传到App Store Connect

1. 登录：https://appstoreconnect.apple.com
2. 选择你的应用
3. 进入"1.0 准备提交"
4. 找到"App 预览和截图"
5. 选择"6.7" Display"或"6.5" Display"
6. 点击"+"上传截图
7. 按顺序拖入5张图片
8. 中英文分别配置

## 💡 技巧

### 获得高分截图的小技巧

1. **多玩几次**，找到容易消除的局面
2. 可以在代码中**临时修改分数**（仅用于截图）：

```swift
// 在 ViewController.swift 的 setupGame() 中临时添加
private func setupGame() {
    gameEngine.delegate = self
    gameEngine.resetGame()
    createGemViews()
    
    // 仅用于截图 - 设置初始分数
    #if DEBUG
    // gameEngine.addScore(300) // 如果有这个方法
    #endif
}
```

3. 截图后记得删除测试代码

## 🎬 替代方案：使用Xcode截图

如果模拟器截图不方便：

1. 在Xcode菜单：**Debug → View Debugging → Take Screenshot**
2. 或使用命令行：

```bash
# 启动模拟器
xcrun simctl boot "iPhone 15 Pro Max"

# 打开模拟器
open -a Simulator

# （在Xcode中运行应用）

# 截图
xcrun simctl io booted screenshot ~/Desktop/screenshot.png
```
