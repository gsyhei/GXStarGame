### 2. App Icon 生成器 🎨
已创建 `AppIconGenerator.swift` 工具类，可生成所有尺寸的App图标。

**如何生成图标：**

在 `AppDelegate.swift` 的 `application(_:didFinishLaunchingWithOptions:)` 方法中添加：

```swift
#if DEBUG
// 仅在调试模式下生成图标
AppIconGenerator.generateIcons()
#endif
```

或者在任意 ViewController 中临时调用：
```swift
AppIconGenerator.generateIcons()
```

运行后，所有图标将保存在 iPhone 的 **Documents 目录**中，Console 会显示路径。

**获取生成的图标：**
1. 在 Xcode 中打开 `Window -> Devices and Simulators`
2. 选择你的设备或模拟器
3. 找到应用，点击下载容器
4. 在容器的 `AppData/Documents/` 目录下找到所有生成的图标

**手动配置图标到项目：**
1. 打开 `Assets.xcassets/AppIcon.appiconset/`
2. 将生成的图标拖入对应的尺寸槽位
3. 确保在 Xcode 中所有图标都已正确配置
