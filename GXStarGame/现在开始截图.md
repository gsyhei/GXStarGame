# 📸 现在开始截图 - 简单5步

## 当前状态
✅ 模拟器已启动  
✅ 游戏已运行

---

## 📸 截图方法

**在模拟器中按键盘：`Cmd + S`**

或者：
- 模拟器菜单 → File → Save Screen
- 截图会自动保存到桌面

---

## 🎯 5张截图清单

### 📸 截图1：主菜单（当前画面）

**如果你现在在主菜单：**
1. 确保看到：
   - 🔮 星辰魔法师标题
   - 💎 跳动的宝石图标
   - 🎮 开始游戏按钮
   - ⚙️ 设置按钮
2. 按 `Cmd + S` 截图
3. 桌面会出现 `Screen Shot 2025-10-12 at XX.XX.XX.png`

✅ 完成第1张

---

### 📸 截图2：游戏初始界面

1. 点击模拟器中的"开始游戏"按钮
2. 进入游戏界面
3. **不要点击任何宝石**（保持初始状态）
4. 确保看到完整的8x8宝石棋盘
5. 按 `Cmd + S` 截图

✅ 完成第2张

---

### 📸 截图3：选中状态

1. 点击任意一个宝石
2. 宝石会放大（选中状态）
3. 保持选中状态
4. 按 `Cmd + S` 截图

✅ 完成第3张

---

### 📸 截图4：游戏进行中（高分）

1. 点击相邻宝石进行交换消除
2. 多玩一会儿，让分数达到 **200-400分**
3. 确保棋盘上有复杂的宝石布局
4. 按 `Cmd + S` 截图

💡 提示：可以多玩几局，选择分数高的时候截图

✅ 完成第4张

---

### 📸 截图5：游戏结束

1. 继续玩直到游戏无法继续
2. 会弹出"🎮 修炼结束"对话框
3. 显示最终能量
4. 按 `Cmd + S` 截图

✅ 完成第5张

---

## 📂 整理截图

### 1. 找到截图文件

桌面上会有5张文件：
```
Screen Shot 2025-10-12 at 10.30.15.png
Screen Shot 2025-10-12 at 10.31.20.png
Screen Shot 2025-10-12 at 10.32.45.png
...
```

### 2. 重命名文件

按顺序重命名为：
```
01-menu.png          (主菜单)
02-game-start.png    (游戏开始)
03-selected.png      (选中状态)
04-gameplay.png      (游戏中)
05-game-over.png     (游戏结束)
```

### 3. 移动到文件夹

```bash
# 移动到专门的文件夹
mkdir -p ~/Desktop/GXStarGame-Screenshots
mv ~/Desktop/Screen\ Shot*.png ~/Desktop/GXStarGame-Screenshots/

# 或手动拖入文件夹
```
