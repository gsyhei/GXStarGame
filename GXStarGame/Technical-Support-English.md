## ğŸ†˜ Frequently Asked Questions

### Gameplay Related

#### Q1: How do I start the game?
**A:** Open the app and you can start playing immediately. No registration or login required.

#### Q2: What are the game rules?
**A:** 
1. Tap to select a gem
2. Tap an adjacent gem to swap
3. Match 3 or more gems of the same color in a line to eliminate them
4. After elimination, gems above will fall down and new gems will be generated
5. Continue matching to get higher scores

#### Q3: When does the game end?
**A:** The game ends automatically when there are no possible moves (no valid swaps can create matches), showing your final score.

#### Q4: How do I restart the game?
**A:** Tap the "ğŸ”„ New Game" button at the bottom of the game interface to start a new game.

#### Q5: How is the score calculated?
**A:** You earn 10 points for each gem eliminated. Chain reactions can earn you more points.

### Technical Issues

#### Q6: What should I do if the game crashes?
**A:** Please try the following steps:
1. Force close the app and reopen it
2. Restart your device
3. Check for system updates
4. Uninstall and reinstall the app (Note: This will clear game records)
5. If the problem persists, please contact technical support with your device model and iOS version

#### Q7: What should I do if the game lags?
**A:** 
1. Close other apps running in the background
2. Ensure your device has enough storage space
3. Check if Low Power Mode is on, recommend turning it off
4. Restart your device

#### Q8: The game won't launch?
**A:** 
1. Confirm your iOS version is 13.0 or higher
2. Check if your device has enough storage space
3. Try uninstalling and reinstalling
4. If the problem persists, please contact us

#### Q9: Game interface displays abnormally?
**A:** 
1. Check if screen rotation is locked (app only supports portrait mode)
2. Try restarting the app
3. If the problem persists, please provide screenshots and contact us

#### Q10: Sound issues?
**A:** The current version does not include sound effects. Stay tuned for future updates.

#### Q11: Where is game data saved?
